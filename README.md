# RSVP Movies Data Analysis Using SQL

## Project Overview

RSVP Movies is an Indian film production company known for producing successful movies for the Indian audience. As part of their expansion strategy, the company plans to release a movie for the global audience and wants to make data-driven decisions.

This project analyzes movie industry data from the past three years using SQL to generate insights and recommendations that can help RSVP Movies plan their upcoming international movie project.

## Problem Statement

The objective of this project is to analyze movie-related datasets and provide actionable business insights for RSVP Movies. The analysis focuses on understanding movie trends, ratings, genres, production companies, directors, actors, and audience preferences.

Using SQL queries, meaningful insights are generated to support strategic decision-making for future movie production and investments.

## Dataset Information

The dataset contains information related to movies released over the last three years, including:

- Movie Details
- Genres
- Ratings
- Directors
- Actors and Actresses
- Production Companies
- Release Dates
- Duration
- Country Information
- Languages
- Financial Performance

## Database Tables

The project uses the following tables:

- movie
- genre
- ratings
- names
- role_mapping
- director_mapping

## Key Analysis Performed

### Movie Analysis
- Total movies released per year
- Month-wise movie trends
- Movies by country and language

### Genre Analysis
- Most popular genres
- Genre-wise movie distribution
- Highest-rated genres

### Ratings Analysis
- Average movie ratings
- Top-rated movies
- Rating distribution

### Director Analysis
- Top directors based on movie ratings
- Director performance comparison

### Actor & Actress Analysis
- Best-performing actors
- Best-performing actresses
- Ranking based on ratings and movie count

### Production House Analysis
- Top production companies
- Production houses with highest-rated movies

## Business Recommendations

Based on the SQL analysis:

- Focus on high-performing genres preferred by global audiences.
- Collaborate with top-rated directors and actors.
- Invest in production houses with a proven track record.
- Target international markets with content backed by audience preferences.
- Use historical ratings and trends to maximize movie success probability.
